📊 E-Commerce Sales Performance & Customer Segmentation Dashboard

🚀 Project Overview

This project analyzes an e-commerce transaction dataset to uncover insights about sales performance, customer behavior, and product demand. Using Python for data preparation and Power BI for visualization, the project transforms raw transactional data into an interactive business intelligence dashboard that supports data-driven decision making.

The solution combines data cleaning, feature engineering, customer segmentation using RFM analysis, and interactive dashboards to provide a complete analytical view of the business.
